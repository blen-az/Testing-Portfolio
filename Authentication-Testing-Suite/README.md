# 🔐 Authentication Testing Suite

## Project Overview
This project contains a comprehensive manual testing suite for standard authentication flows, including **Registration, Login, Password Reset, Session Management, and Account Lockout**.

These flows are critical to modern web applications, and testing them involves a mix of **Functional, Negative, and Security testing**. 

This suite acts as a demonstration of mid-level QA testing methodologies for standard authentication systems.
